# 杭州市上城区片区分类工具

## 快速启动

先设置高德 Web 服务 API Key（不要把 Key 写进代码或前端）：

PowerShell：

```powershell
$env:GAODE_KEY = "你的高德Web服务Key"
& "<Python路径>" app.py --port 5050
```

然后打开 `http://localhost:5050`。

可访问 `http://127.0.0.1:5050/healthz` 检查高德服务配置；其中 `gaode_configured` 应为 `true`。
如果结果显示“网络错误”，请检查电脑是否能访问高德接口或是否被防火墙拦截；如果显示“Key无效”或“调用额度不足”，请到高德控制台检查 Key 权限和配额。

公开部署前请为高德 Key 设置接口白名单和调用额度，并使用 HTTPS、限流和反向代理。

## 当前能力

- 地图绘制、编辑、保存和加载片区；片区自交、重复命名和重叠会被拒绝。
- 上传 `.xlsx`，单文件最多 5000 行、20MB。
- 地址解析采用后台任务和进度轮询，结果带置信度、定位来源和复核状态。
- 低置信度、无法定位、片区外和边界附近记录进入人工复核。
- 复核支持地图标记拖拽、片区选择和备注，导出时保留原始列并追加分类字段。
- 任务只保存在内存中，任务过期或下载后会清理。

## 功能概述
根据Excel中的公司地址信息，自动划分到对应的片区。支持：
1. 上传Excel文件（包含公司名称和地址）
2. 基于高德地图API的地理编码（地址→坐标）
3. 在地图上绘制/调整片区多边形
4. 自动将坐标分类到对应片区
5. 导出分类结果到Excel

## 使用前提
- Python 3.7+
- 高德地图Web服务API Key（已配置）

pip3 install flask openpyxl requests

`ash
python3 app.py
`
打开浏览器访问 http://localhost:5050

## 使用说明
### 第一步：定义片区
1. 用地图左侧绘图的工具栏绘制多边形
2. 输入片区名称，选择颜色
3. 点击添加当前绘制为片区
4. 保存片区配置

### 第二步：上传数据
1. 上传Excel文件（.xlsx格式）
2. 选择公司名称列和地址列

### 第三步：地理编码
1. 点击地理编码按钮
2. 系统通过高德API将地址转为坐标

### 第四步：分类与导出
1. 系统自动将坐标点归入对应片区
2. 查看分类结果
3. 导出为Excel文件

## 文件结构
- app.py - Flask后端服务
- templates/index.html - 前端地图交互界面
- zones.json - 片区数据持久化文件
- static/ - Leaflet地图库文件
- requirements.txt - Python依赖
- start.sh - 统信UOS启动脚本
